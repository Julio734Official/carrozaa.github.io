Carroza de Duolingo

Cortar

Se dibujó dos diseños distintos de un árbol en el cartón, después se los cortó al medio.

Armar

Luego se los encastró, pero para que no se desarmen, se les pegó tiritas de cartón en forma de L a las cuatro caras y en la base para que no se caiga.

Forrar

Después se los forró con tiras de papel crepé desde la base del árbol y hasta las putas de cada rama.

Detalles finales

Para finalizar, se pegó con papel crespe que hacian de las hojas del árbol.
