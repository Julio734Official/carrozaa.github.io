¿Qué es Duolingo?

Duolingo es una plataforma en línea y aplicación móvil que se ha convertido en la forma más popular para aprender idiomas de manera gratuita.

Armado de la carroza

Aquí demostraremos como armamos la carroza de principio a fin. Algunas secciones contiene más información del proceso.

Chasis

Para realizar el proyecto, primero hicimos el chasis, utilizando madera para realizar la forma y tornillos para unirlos y asegurarlos. El chasis tenía como medidas de un metro de largo y 60 centimetros de ancho.

Forrar el Chasis

Forramos el chasis con carton, usando Fortex para pegarlo con el chasis.

Decorar la Carroza

Era el momento de decorar la carroza. Haz click en el título para ver el proceso de la decoración

Árboles

Haz click en el título para ver el proceso del armado de los árboles.

Van

Para la van se desarmaron tres cajas de zapatos para poder hacer la estructura, luego se dibujó los detalles que debían recortarse, como las ruedas y las ventanas. Después se pegó las partes con pegamento para poder armar la van y, una vez armada, se procedió con los detalles finales, como pintarla.
